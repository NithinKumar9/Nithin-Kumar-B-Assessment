# Assessment 1

# Medical & Travel Expense Request 

## Overview

This is a Medical & Travel Expense Request form. It is used by workers to claim reimbursement for expenses they have paid during their medical treatment after a workplace injury.

The form contains different sections such as prescription medicines, medical supplies, parking charges, travel mileage, and bus or taxi expenses. The user enters all the required details, and these are used for reimbursement processing.

### Technologies used

- HTML
- CSS
- JavaScript

# Assessment 2

# Worker Progress Report 

## Overview

This is a Worker Progress Report form. It is used to collect updates from an injured worker during their recovery process. The worker provides information such as whether they have returned to work, their current recovery status, pain level, ongoing treatment, medication, home exercises, and any additional comments.

- CSS
- JavaScript
